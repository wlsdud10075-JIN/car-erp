영업 업무 가이드영업 · v2

문의 → 응대 → 매입보드 등록(엔카/경매) → 검차팀 현지확인·최종금액 → 바이어 → 경매/구매 → [관리]
바이어가 채팅앱으로 문의하면 영업이 직접(영어로) 응대해서, 그 차를 매입보드에 등록(엔카 또는 경매)합니다. 검차팀이 현지에서 실차를 보고 최종금액을 산정하면, 영업이 사진과 최종금액을 바이어에게 전달하고, 수락되면 경매/구매 후 [관리]에게 카톡 전달합니다.
※ 현재 기준(자동 AI 응대·자동 연동 도입 전). 단계마다 담당이 다릅니다 — 아래 색으로 구분.

📋 단계별 상세

1
채팅 문의 접수 & 1차 응대
영업
할 일 — 영어로 응대 (바이어 대부분 해외)
  1. 채팅에 차 정보가 자동으로 옵니다 — 매물번호·차종·연식·주행거리·가격·사진. 매물번호(c_no)를 메모.
  2. 재고·기본사항 안내하고, "실차를 확인한 뒤 사진과 최종금액을 보내드리겠다"고 안내 (가격은 현지 확인 후 확정).
  3. 그다음 2단계(매입예정 등록)로.
💬 1차 대화 (English / 한글 자막)
🧔🏽 바이어
Hi SSANCAR, is this Genesis GV70 (#6797296) still available?
안녕하세요, 이 제네시스 GV70 (#6797296) 아직 있나요?
👨‍💼 영업
Hello! Let me check availability — the 2022 GV70 2.5T, 41,189 km, right? We'll inspect the actual car and send you photos with the final price.
안녕하세요! 확인해 드릴게요 — 2022년식 GV70 2.5T, 41,189km 맞으시죠? 실차 확인 후 사진과 최종금액을 보내드리겠습니다.
🧔🏽 바이어
Great. How long will it take?
좋아요. 얼마나 걸릴까요?
👨‍💼 영업
Usually within a few hours — I'll message you with photos and the final price.
보통 몇 시간 내예요 — 사진과 최종금액으로 다시 연락드릴게요.
↳ 매입예정 등록 + 검차팀 현지확인으로
⚠️ 주의
"아직 있나요?"는 원본 사이트(엔카/ssancar.com) 재고 기준으로 답. 최종금액은 현지 확인 전에 확정해 주지 말 것 (차 상태로 달라짐). 큰 할인 약속은 윗선 확인.
2
매입예정 등록 (매입보드) — 엔카 / 경매 선택
영업
할 일 — [+ 매입예정 추가] → 먼저 출처 선택
🛒 엔카 (즉시구매)
  • 엔카 매물 URL / 매물번호 붙여넣기 (식별용)
  • 판매 딜러 / 지역
  • 시간 마감 없음 — 상시 등록
🔨 경매
  • 경매장 / 출품번호
  • 10:00 매입예정 마감 (이후 [관리]만 수정)
공통 입력
  • 차량번호 · 차대번호(VIN) — 필수 (등록 후 수정 불가, 중복 방지 키)
  • 예상가 — 선택 (엔카/경매 매물 표시가를 넣어두면 좋지만 비워도 됨. 진짜 금액은 현지 확인 후 확정)
⏰ 시간 마감 — 경매만
경매 차량은 10:00 이후 잠금(관리자만 해제). 엔카는 상시라 마감 없음. 본인 등록 차량만 본인 화면에 보임.
3
현지 차상태 확인 & 최종금액 산정
검차팀
검차팀이 하는 일 (영업은 결과 대기)
  1. 현장에서 실차 상태 확인 (외관·주행·사고흔적 등) + 외관 사진 촬영.
  2. 차 상태를 반영해 현지 최종금액 산정 (예상가와 달라도 됨 — 실제 상태가 기준).
  3. 최종금액·사진을 보드에 입력 → 영업에게 넘어감.
⚡ 현지(한국) 흥정은 실시간으로 빠르게 진행됩니다. 검차팀이 차 보고 금액 정하는 건 보통 그 자리에서.
🛡 개인정보
촬영·공유 사진은 차량 외관/상태만. 번호판·서류(등록증·성능지·말소신청서)는 제외.
4
바이어에게 사진 + 최종금액 전달 → 회신
영업
할 일
  1. 검차팀이 올린 실차 사진 + 최종금액을 바이어에게 전달.
  2. 바이어 회신을 기다림 — 수락 / 거절. 흥정으로 금액이 바뀌면 최종금액을 다시 수정.
  3. 수락되면 5단계(경매/구매)로.
💬 2차 대화 (사진 + 최종금액)
⏱ 몇 시간 뒤 — 바이어 응답은 늦을 수 있음
👨‍💼 영업
📷 Here are the actual photos. After on-site inspection: clean record, minor scratch on front bumper. Final price: 13,200,000 KRW. Shall we proceed?
📷 실차 사진입니다. 현지 확인 결과 무사고, 앞범퍼 미세 스크래치. 최종금액 13,200,000원. 진행할까요?
🧔🏽 바이어
Looks good. Yes, please proceed!
좋네요. 네, 진행해 주세요!
↳ 수락 → 경매/구매로 (거절이면 종료)
⏱ 바이어 대화는 몇 시간 뒤 시작되거나 몇 시간 이어질 수 있습니다. "회신대기"로 두고 다른 차량을 먼저 처리하세요.
5
경매 · 구매 (수락 차량만)
경매팀 영업
🔨 경매팀이 하는 일
  • 바이어 수락 차량만 진입 (미수락 차단)
  • 현지 최종금액으로 집행 — 경매=낙찰/유찰, 엔카=구매확정/취소
  • 낙찰·구매확정 = SSANCAR 소유(우리 재고)
👨‍💼 영업이 하는 일
  • 결과 확인
  • 유찰/취소 시 바이어에게 안내
  • 낙찰/구매확정 시 바로 6단계
6
★ [관리]에게 카톡 전달 (핵심 인계)
영업 → 관리
할 일 — [관리]에게 카카오톡으로 아래 전달 → [관리]가 car-erp 등록
📤 카톡에 넣을 정보 체크리스트
⚠️ 꼭 지킬 것
매물번호(c_no)를 같이 — 처음 문의~거래완료까지 한 줄로 이어집니다. 누락 없이 한 번에 보내면 [관리] 재입력이 줄어듭니다.

👥 역할별 요약

👨‍💼 영업
  • 채팅 응대(영어)
  • 매입예정 등록(엔카/경매)
  • 바이어에 사진+최종금액 전달
  • 유찰/취소 안내
  • [관리] 카톡 전달
🔍 검차팀
  • 현지 실차 검수
  • 외관 사진 촬영
  • 차상태 반영 최종금액 산정
🔨 경매팀
  • 수락 차량 진입
  • 최종금액으로 집행
  • 경매=낙찰/유찰
    엔카=구매확정/취소

⚠️ 자주 하는 실수

  1. 현지 확인 전에 바이어에게 최종금액을 확정해 줌 → 차 상태로 금액 달라짐, 신뢰 문제.
  2. 경매 차량 10:00 마감을 넘겨 등록 못 함 (엔카는 상시).
  3. VIN 누락 → car-erp 등록 시 중복·오류. 등록 후 수정 불가니 처음에 정확히.
  4. 바이어에게 서류·번호판 사진 전송 → 금지(개인정보).
  5. [관리] 카톡에 매물번호(c_no)·출처 빠뜨림 → 추적 끊김.
