수출통관 업무 가이드수출통관

판매완료 인수 → 반입(선적) → 통관 → B/L → DHL → 거래완료
영업이 판매를 마친 차량을 받아 반입(선적)·수출통관·B/L 발급·DHL 발송까지 처리해 거래완료로 만드는 절차입니다. car-erp 차량 편집의 수출통관 / 선적(B/L) / DHL / 서류 탭에서 진행합니다.
※ 수출(export) 채널 차량만 해당. 헤이맨·카풀(국내)은 통관·B/L·DHL 흐름이 없습니다. (현재는 [관리]가 이 업무도 겸함)

🚦 진행 게이트 (입금률 조건 — 꼭 기억)

입금이 얼마나 됐는지에 따라 다음 단계 진입이 막힙니다
50% 이상 입금률 50% 넘으면 → 반입(선적)·통관 진입 가능
100% 완납 잔금까지 100% 완납돼야 → B/L 발급 가능 (미완납이면 [관리]/관리자 승인으로만 우회)

📋 단계별 상세

1
판매완료 차량 인수
판매·입금 50%↑ 후
할 일
  1. 판매가 등록되고 입금률 50%를 넘은 수출 차량을 통관 대상으로 확인.
  2. 차량 편집 패널 → 수출통관 탭으로 이동.
2
통관 정보 입력 (수출통관 탭)
반입 전·후
할 일 — car-erp 수출통관 탭
항목설명
통관 바이어 / 컨사이니실제 통관 서류상 당사자필수
포워딩사운송 대행사 지정필수
면장금액 (USD)수출신고 금액 (매출 검증용)
도착일자(ETA)·RORO/CONTAINER운송 방식·예정일
Port of Loading선적항
수출신고서 업로드신고 완료 서류 첨부
3
반입(선적) 처리 (선적 B/L 탭)
반입지 입력 시점
할 일
  1. 반입지(선적지)를 입력하면 진행상태가 "선적중"으로 넘어갑니다.
  2. 선적 바이어/컨사이니, 선적일, VSL(선박) 정보 입력.
📄 선적 서류 생성 — 서류 탭에서 RORO/컨테이너 Invoice&Packing·Contract 자동 생성. 여러 대를 한 서류에 담으려면 차량목록에서 체크 후 일괄 생성.
4
B/L 발급
100% 완납 후
할 일 — 선적(B/L) 탭
  1. B/L번호, 컨테이너 No, VSL 입력.
  2. B/L 문서 업로드 → 진행상태가 "거래완료"로.
⚠️ 100% 완납 게이트
B/L 발급은 잔금까지 100% 완납돼야 가능. 미완납 차량은 [관리]/관리자 승인(우회) 없이는 막힙니다. 임의 발급 금지.
5
DHL 발송
B/L 후
할 일 — DHL 탭
  1. 수취인·발송인 정보, 중량/크기 입력.
  2. DHL 발송신청 체크 → 추적번호 기록(수동).
6
통관 SET 서류 / 마무리
필요 시
📄 통관 SET — 서류 탭에서 8시트 통관 SET 생성(구매리스트 마스터→6시트 자동연동). 매매업등록번호 등 NICE 미제공 항목은 공란→수기.
  1. 모든 서류·B/L·DHL 완료 → 거래완료 상태 확인.

⚠️ 자주 하는 실수

  1. 50% 미만인데 통관 진입 시도 → 막힘. 입금률 먼저 확인.
  2. 100% 미완납인데 B/L 발급 시도 → 승인 없이는 불가.
  3. 국내(헤이맨·카풀) 차량에 통관/B/L 처리 시도 → 수출 채널만 해당.
  4. 포워딩사·통관 바이어 누락 → 서류 생성 시 공란.
